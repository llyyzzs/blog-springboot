# 个人博客后端

推荐使用docker，下载MySQL、RabbitMQ、Minio、Redis、ElasticSearch

配置application.yml

```
运行sql文件，将MySQL数据库连接信息以及Redis填写成自己的
配置自己的rabbitmq地址、账号
配置自己的elasticsearch地址
注册自己的邮件用于发送邮件，配置自己的mail
配置自己的Minio或OSS
```
